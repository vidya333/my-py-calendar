# ⚡ Epic Life Calendar

An interactive timeline application designed to help you track your wins, monitor your personal growth habits category-wise, and visually curate your year.

## ✨ Features
- **Interactive Calendar:** Click any day to log an achievement or milestone instantly.
- **Dynamic Category Mapping:** Visual tags automatically color-code tasks by category.
- **Analytics Dashboard:** Real-time generation of graphs tracking consistent output.
- **Aesthetic Customization:** Custom cover file uploads mapped directly to individual months.

## 🛠️ Stack
- **Framework:** Streamlit
- **Visualization:** Plotly Engine
- **Calendar API:** FullCalendar wrapper (`streamlit-calendar`)
- **Data Structuring:** Pandas

## 🚀 Quickstart
```bash
pip install -r requirements.txt
streamlit run app.py
